-p 2000-3000 明确命令 Nmap：“只去扫描 2000 到 3000 这 1001 个指定的端口”。
-F（Fast scan）在 Nmap 里的意思是：“只扫描全量端口中最常用的前 100 个高频端口（比如 80, 22, 443 等）”。
-P 和 -F 不能同时使用。
--max-retries 1：限流保护。如果某个端口由于防火墙拦截没有回应，Nmap 最多只重试 1 次，绝不在短时间内疯狂重发垃圾包，完美避开了平台的 DoS 熔断机制。
-v：（详细模式）只要发现一个开放端口，立刻在屏幕上打印出来，不需要等整个扫描全部结束，让你实时掌握敌情。
-sS -sV -O：继续保持我们大纲范围内的：SYN 隐蔽扫描、软件版本精准识别、操作系统特征强行刺探。

A Faster alternative for the future: Python HTTP Server
If you ever get tired of uploading to GitHub just to download it on the same physical computer, here is an industry-standard shortcut you can run directly inside your Kali terminal.

In your Kali terminal, navigate to the directory where your scan files are saved and run this:

Bash


python3 -m http.server 8080
This instantly turns your Kali machine into a temporary web server. Now, you can just open Edge on your host computer, type in your Kali machine's IP address followed by the port (e.g., http://192.168.1.x:8080), and you will see a list of all your scan files ready to click and download instantly—no internet upload required!
